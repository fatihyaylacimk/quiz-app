# 🎯 Quiz App

A full-stack dynamic quiz application with real-time math question generation.

## 🚀 Features

- 🔢 Dynamic math questions (auto-generated)
- 🎯 Multiple choice answers
- ⚡ Instant feedback (correct / wrong)
- ⏱ Timer system
- 🎮 Difficulty levels (Easy / Hard)
- 💎 XP system
- 🏆 Rank system (Bronze / Silver / Gold)
- 📊 Leaderboard with Chart.js
- 🌙 Dark mode
- 📱 Mobile responsive design
- 🎨 Animations (no external assets)

## 🛠 Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Chart: Chart.js

### Backend
